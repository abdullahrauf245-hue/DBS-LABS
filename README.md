# CS-212: Object Oriented Programming (OOP) Lab

This repository contains my laboratory work and programming assignments for the **CS-212: Object Oriented Programming** course at **NUST**.

## 🚀 Project Overview
The focus of this repository is strictly on implementing core OOP concepts using **Java**. Each directory or file represents a specific lab task or concept explored during the semester.

## 🛠️ Concepts Covered
* **Encapsulation:** Using private fields and public getters/setters.
* **Inheritance:** Extending classes to reuse code.
* **Polymorphism:** Method overriding and overloading.
* **Abstraction:** Working with abstract classes and interfaces.
* **Exception Handling:** Managing runtime errors gracefully.
* **File I/O:** Reading from and writing to files in Java.

## 💻 Tech Stack
* **Language:** Java
* **Tools:** VS Code / IntelliJ IDEA
* **Version Control:** Git
